This page shows whether your current browser has native support for view transitions.

|Feature|Support?|
|-------|---------|
|View transitions Level 1|<span id="level1"/>|
|View transitions Level 2|<span id="level2"/>|

Level 1 is the basic view transition API for transitions within th same document.\
Level 2 covers cross-document view transitions.

[Pointers to the specs](/jotter/api/references/#w3c-drafts) (W3C Drafts)



Browser compatibility data: https://developer.mozilla.org/en-US/docs/Web/API/View_Transitions_API#browser_compatibility
